# Carniceria

La idea general es implementar un servicio de pedidos (MAYORISTAS) para un sistema de carniceria donde podamos hacer pedidos por kilos con nuestros productos en una base de datos donde tenemos los parametros que despues mandaremos mediante Flask, consumiendolas en Js mediante una API

## Problematica
Tendremos un nav bar donde podremos navegar a los productos al carrito de compras justo abajo todos los productos de la marca el cual estan mostrados como cuadros el cual tendran una pequeña descripcion junto al precio por kilo. Al ingresar a cada producto se debera mostrar una imagen ilustrativa junto con descripcion acerca del producto, de donde proviene, el tipo de res y para que es usado este corte

## Tecnologias implementadas
Recordemos que esta aplicación es fullStack, por lo que separamos tecnologias
### Backend
- Python y Flask - utilizada para manejo de rutas y el backend en general
- MySQL - Guardar la informacion de los productos junto con el stock y toda la informacion necesaria
### FrontEnd
- Html - Etiquetas
- Css - estilos
- JavaScript - consumir las APIs
